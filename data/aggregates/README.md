# Aggregated, anonymised data

This folder is the **public, citable** companion to the monograph:

> Kóňa, A., & Cagáňová, D. (2026). *Digital Habits and Smart-City Readiness of Secondary-School Students.* Springer Nature.

It contains **only sample-level aggregates** — descriptive statistics, correlation matrices, group means, cluster profiles and the municipal Smart-People composite. **No individual-level rows are included**, so the files can be re-distributed without breaching the school-level data-protection commitments described in §4.3 of the monograph.

The full ethical chain: questionnaires were anonymous from the start; school management informed students/guardians; participation was voluntary; only aggregate results are released.

## Reproduction

These files are regenerated by `scripts/08_export_aggregates.py` from the raw surveys (kept locally per `data/README.md`). To rebuild, place the three source files in `data/`, then:

```
python scripts/01_build_indices.py
python scripts/04_clusters.py
python scripts/05_sensitivity_pca.py
python scripts/06_bootstrap_ci.py
python scripts/08_export_aggregates.py
```

## What is intentionally NOT here

- Individual responses from either survey
- Free-text fields (open-ended motives, app names, city names)
- School identifiers
- Any field that could re-identify a respondent

For reasoned academic replication of the **individual-level** analyses, contact the corresponding author with a brief data-use description (see `data/README.md`).
